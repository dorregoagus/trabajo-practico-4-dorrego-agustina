# Trabajo Práctico 4 - API REST de Películas

**Materia:** Taller de Lenguaje de Programación I  
**Año:** 2026

## Integrante

- Dorrego Agustina

## Descripción

Desarrollo de una API REST para la gestión de películas utilizando Node.js, Express, Sequelize y MySQL.

## Funcionalidades

- Obtener todas las películas.
- Obtener una película por ID.
- Crear una película.
- Actualizar una película.
- Eliminar una película.

## Tecnologías utilizadas

- Node.js
- Express
- Sequelize
- MySQL
